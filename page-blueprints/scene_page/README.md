## More_page for Dwains Dashboard v3
##### Created by Bourner
---


### Installation: 
  
1.  Go to More Page on DD3
2.  Create a new more page
3.  Use Dwains Dashboard Blueprint
4.  Add YAML Code from file to HA
5.  Click install Blueprint
6.  Click Use this Blueprint
